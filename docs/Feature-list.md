# Naksha Feature List
AI-Powered Astrology App for Personalized Guidance and Spiritual Mapping

## MVP Feature Set (V1.0 Launch)

### User Authentication & Profiles
- Sign up and log in via email/password (JWT-based sessions)
- Edit user profile: name, birth info (DOB, TOB, location), time zone
- Store preferences (notifications, chart style, zodiac system)

### Natal Chart Engine
- Generate natal chart based on user birth data
- Visual astrology chart wheel (interactive UI: planets, signs, houses, aspects)
- List view of natal placements (e.g., Sun in Virgo, Moon in Scorpio)
- Save and label charts (support for multiple charts)

### AI-Powered Guidance (GPT-4o Integration)
- Ask-Astrologer chat interface
- Smart prompt routing (natal reading, shadow work, love life, life purpose)
- Generate daily guidance based on current transits
- Interpret transits relative to the user’s chart
- Save and revisit AI conversations

### Daily & Weekly Transits
- Personalized daily horoscope (Sun, Moon, Rising focus)
- Weekly forecast with key themes and suggestions
- Push notification system for new forecasts, retrogrades, eclipses
- “Today’s Energy” dashboard: mood, warning, opportunity

### Saved Readings & History
- Save chart interpretations and AI readings
- Journal integration (write reflections or responses to AI prompts)
- View reading/archive history by date or topic
- Tag and favorite specific readings

### Shadow Work & Self-Development
- AI-generated journal prompts based on transits or natal placements
- Daily or weekly introspection cycles
- Guided practices (rituals, affirmations, meditations – basic at first)
- Track shadow work completion or journaling milestones

## Post-MVP / Stretch Features

### Synastry & Compatibility Tools
- Create a second chart (e.g. partner or friend)
- Generate compatibility report (synastry + composite)
- AI interpretation of relationship dynamics

### Monetization: Reports, Cosmetics & Subscription
- Deep Natal Report (longform, downloadable)
- Personalized Transit Timeline (6–12 month view)
- Cosmetic Upgrades: chart themes, skins, astrology art (in-app purchase)
- Subscription Tiers:
  - Pro ($4.99): unlimited chat, saved reports
  - Elite ($9.99): early access, deeper coaching, AI rituals

### Calendar & Notifications
- Transit calendar integration
- View upcoming aspects, retrogrades, eclipses
- Sync to Google or Apple Calendar
- Notification settings by planet/sign/intensity

### Community & Social
- Share chart or AI insights with others (text/image export)
- Public feed of “Today’s Energy” reactions
- Optional: Comment/react (private mode always available)

### AI Spiritual Coach (V2+)
- Daily coaching check-ins based on past charts/journals
- Progress tracking (karmic themes, goal alignment, repeating transits)
- Personalized rituals and mantra guidance
- Ask for custom meditations, invocations, altar prep

## Internal / Infrastructure

### Admin Tools
- User Logs / Bug Reports: View user activity, error logs, AI failures

### Analytics
- Usage Metrics: Track feature usage, dropoff, user growth

### AI Monitoring
- Prompt Audit Logs: Store prompt/response for QA & LLM fine-tuning
